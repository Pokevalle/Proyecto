ParkPlus - Sistema de Gestión de Parqueo

ParkPlus es una aplicación de escritorio desarrollada en Java utilizando
NetBeans, JDBC y MySQL.
El sistema permite controlar el ingreso, salida, reingreso y cobro de
vehículos, así como la administración de usuarios, reportes diarios y
visualización del mapa de parqueo.

Características principales

-   Registro de personas (Estudiante, Catedrático, Visitante)
-   Manejo de tickets (Flat y Variable)
-   Salidas y salidas temporales
-   Reingreso con validaciones automáticas
-   Reportes diarios de ingresos
-   Mapa visual del parqueo con áreas y estados de cada spot
-   Base de datos MySQL integrada

Requisitos

-   Java 8 o superior
-   NetBeans 8+
-   MySQL Server
-   Conector JDBC

Ejecución

1.  Importar el proyecto a NetBeans.
2.  Crear la base de datos incluida en el archivo del proyecto.
3.  Ajustar la conexión en dbConnection.java.
4.  Ejecutar la clase prueba (ventana principal).
